# Setup e ambiente di sviluppo

I passi tipici per avviare il progetto in locale:

1. Installa dipendenze

   npm install

2. Compila gli asset CSS (se necessario)

   npm run build:css

3. Avvia il server in sviluppo

   npm run dev
